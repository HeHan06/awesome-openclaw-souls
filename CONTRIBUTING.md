# 🤝 贡献指南

感谢你对 **Awesome OpenClaw Souls** 的兴趣！我们欢迎所有形式的贡献。

---

## 📋 贡献方式

### 1. 提交新的 Soul 风格

发现或创建了一个很棒的对话风格？分享出来！

### 2. 改进现有风格

发现某个 soul 可以更好？欢迎优化！

### 3. 修复问题

发现 typo 或错误？帮忙修正！

### 4. 完善文档

让文档更清晰、更易懂。

---

```bash

```
